## 🛠️ Tech Stack
* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas
* **AI Integration:** Google Gemini API
* **Email Service:** Nodemailer (Gmail SMTP)

---

## 🚀 Key Features of SevaSync AI
1. Core Functionality
* Centralized Community Dashboard: A real-time interface where volunteers and admins can track ongoing community tasks (e.g., "Planting trees in the university courtyard").
* AI-Powered Insights: Integrated with Google Gemini AI to provide deeper context and guidance for community needs, making volunteering more "informed".
* Task Management System: Full CRUD (Create, Read, Update, Delete) capability for tasks, allowing the community to dynamically respond to new needs.

2. Security & User Management
* Multi-Role Authentication: Support for different user types, such as "NGO Admin" or "Volunteer," ensuring proper permission levels.
* Smart Password Recovery: A professional backend flow that generates secure tokens, saves them to MongoDB, and emails them to users via Nodemailer.
* Email Automation: Integration with Gmail SMTP to provide real-time updates and account security notifications.

3. Unique Safety Features
* SOS Emergency Trigger: A dedicated "3s for SOS" button that allows users to instantly signal for help in high-risk environments.
* Live Server Monitoring: A health-check system that ensures the backend is always "awake" and connected to the database before users start interacting.
